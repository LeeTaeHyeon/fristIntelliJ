# IntelliJ 단축키

1.	ctrl + shift + A : action창  
	shift x2 : all창

2.	alt + 1 : 프로젝트 네비게이션 이동  
	alt + insert : 새로운 파일 생성  
	esc : 코드창으로 복귀

3.	ctrl + D : 라인 복사  
	ctrl + Y : 라인 삭제  
	alt + shift + 방향키 : 라인이동  

4.	ctrl + K : commit  
	ctrl + shift + K : push  
	action에서 pull : pull  

5.	shift + f10 : run  
	ctrl + f2 : stop
